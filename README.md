# Nexosis API Dashboard

Interact with the Nexosis API with a dashboard.

## Get It Running

**1.** Run npm Install

> npm install 

**2.** Set url and api key:

Create a file, config.json.  Provide your api key.

``` JSON
{
    "apiKey": "My Key",
    "url": "https://ml.nexosis.com/v1"
}
```

**3.** Build and run:

> npm run open-app
